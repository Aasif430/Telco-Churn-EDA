# Telco Customer Churn - Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Telco Customer Churn dataset to understand customer behavior and identify factors influencing churn.

## Objective
To analyze customer data and discover patterns that contribute to customer churn.

## Key Analysis Performed
- Data Cleaning
- Handling Missing Values
- Univariate Analysis
- Bivariate Analysis
- Churn Distribution Visualization
- Feature-wise Churn Comparison

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Insights
- Identified important features affecting churn
- Observed churn trends based on tenure and charges
- Found service-related factors influencing churn

## Files Included
- Telco_Churn_Analysis.ipynb
- customer_churn.csv
